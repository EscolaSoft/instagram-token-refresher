# Instagram Token Refresher

Serwis odświeżający regularnie `LongLived Access Tokeny` instagrama

## Installation

```bash
$ yarn
# or: npm install
```

## Running the app

```bash
# development
$ yarn start
# or: npm run start

# watch mode
$ yarn start:dev
# or: npm run start:dev

# production mode
$ yarn start:prod
# or: npm run start:prod
```

### Watching scss

Auto scss recompilating

```bash
$ yarn scss:watch
# or: npm run scss:watch
```

## Deployment

```bash
# build the app
$ yarn build

# start in production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
